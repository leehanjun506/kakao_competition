add - 새로운 기능 혹은 코드를 추가할 경우  
style - 스타일 추가 혹은 변경  
fix - 기존코드에 버그수정 및 수정 된 기능이 있을 경우  
rename - 파일이름 및 명시된 이름이 변경 되었을 경우  
remove - 코드가 삭제 된 경우 
delete - 파일이 삭제된 경우  
refactor - 코드를 전면수정한 경우  
move - 파일이 다른 디렉토리로 이동했을 경우  
test - 테스트 콛, 리펙토링 테스트 코드 추가  
chore - 빌드 업무 수정, 패키지 업무 수정

```
# commit 예시
add - 새로운 기능 추가
```
